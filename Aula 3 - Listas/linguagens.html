Lingua de Programação

Python

Python é uma linguagem de programação de alto nível,
amplamente usada em aplicações da Web, desenvolvimento
de software, ciência de dados e machine learning e fácil de ler
e escrever.

JavaScript

JavaScript é uma linguagem de programação de script que
permite interatividade em páginas web.

Java

Java é uma linguagem de programação orientada a objetos
utilizada para desenvolvimento de aplicações.

C++

C++ é uma linguagem de programação de propósito geral,
derivada de C, amplamente usada para desenvolvimento
de software.
